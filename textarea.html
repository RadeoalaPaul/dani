Eticheta textarea definește un control de introducere a textului pe mai multe linii.
Elementul textarea este adesea folosit într-un formular, pentru a colecta intrări ale utilizatorilor, cum ar fi comentarii sau recenzii.
O zonă de text poate conține un număr nelimitat de caractere, iar textul este redat într-un font cu lățime fixă ​(de obicei, Courier).
Mărimea unei zone de text este specificată de atributele cols și rows (sau cu CSS).
Atributul nume este necesar pentru a face referire la datele formularului după ce acesta este trimis (dacă omiteți atributul nume, nu vor fi trimise date din zona de text).
Atributul id este necesar pentru a asocia zona de text cu o etichetă.
