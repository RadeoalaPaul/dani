Elementul select este folosit pentru a crea o listă derulantă.
Elementul select este cel mai des folosit într-un formular, pentru a colecta input-ul utilizatorului.
Atributul nume este necesar pentru a face referire la datele formularului după ce acesta este trimis (dacă omiteți atributul nume, nu vor fi trimise date din lista derulantă).
Atributul id este necesar pentru a asocia lista drop-down cu o etichetă.
Etichetele option din interiorul elementului select definesc opțiunile disponibile în lista derulantă.
